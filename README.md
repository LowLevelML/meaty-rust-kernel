# Thamognya's Meaty Rust Kernel

This is a kernel made by me following the tutorial [blog_os](https://github.com/phil-opp/blog_os).

## Commands

### To run with qemu:

```console
cargo run
```

### to install targets

```console
rustup target add x86_64-unknown-linux-gnu # or any other target
```
